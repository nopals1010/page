Featured Project
Acintya: Automated Operations Platform

Built a production SaaS from scratch to replace all spreadsheet-based workflows for a live education business. Attendance marking triggers automatic credit deduction, which triggers invoice creation via a billing API, which triggers a WhatsApp notification to the parent. Zero manual steps. The system runs 11 rule-based automations, an append-only audit ledger, role-based access for admin and teachers, and an AI-generated parent summary pipeline via Groq. Designed with an agent-ready architecture as Layer 1 of a broader AI Operating System.
